녹음 기반 스펙트럼 분석기

녹음한 소리를 웹 메모리에 임시 저장한 뒤 스펙트로그램으로 전개하고, y축 드래그로 정한 컷오프 주파수 \( y \) 에 대해 0 ~ y Hz 성분만 남겨 다시 재생한다.

시간 →
진동수 ↑

현재 상태

샘플링 주파수
-
오디오 길이
-
최대 분석 주파수
-
재생 컷오프
-
녹음을 시작해라.

분석 설정

색상 해석

검을수록 약하고, 밝고 따뜻한 색일수록 진폭이 큰 영역이다.

조작 방식

1. 녹음 시작 → 녹음 종료
2. 스펙트럼 전개
3. 그래프 왼쪽 핸들을 y축 방향으로 드래그
4. 그 위치의 주파수값을 \( y \) 라 할 때, 재생은 0 ~ y Hz 만 남겨서 한다.
5. 원본 재생과 필터 재생을 비교하면 된다.
